# Calendar Event Generator

## Usecase: event generator from calendar and popups
1. Type or speak within calendar page 
2. Create plans for event creation
    1. Approve
    2. Retry
    3. Reject
3. Proceed and Create event(s)

## Features
- [ ] Display overlay popup in calendar page
- [ ] Display type or speak to create event
- [ ] Display Chrome built-in AI model download button on Chrome if not installed
- [ ] Use Chrome built-in AI model to generate event creation plan(s)
- [ ] Display event creation plan(s) in overlay popup with google calendar event preview 
- [ ] Display Approve, Retry, Reject event creation plan(s) in overlay popup
- [ ] Proceed and Create event(s) if approved
